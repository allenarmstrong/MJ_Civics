Self Check: Answers Text Version

Now, let's make sure you are ready to navigate the course.

  • What button would you select to go back to the course home page?
    You can select the "Back to Home" link at the top left or the house icon at the top right.
  • Select the glossary term to see its definition.
    Each glossary term is a turquoise color and has a dotted underline.
  • What would you select for the fastest navigation to another unit?
    You can select the Units drop-down list located at the top middle of the course.
  • Select the arrow that will move you to the next lesson page.
    Choose the right arrow located at the top right of the course.
  • Select the spot that will begin the activity on this page.
    All activities have a "Begin," "Go," "OK," "Continue," or similar button to start the activity.
  • Select the link to your note-taking file for the lesson.
    Select the link "CAP File" to access note-taking help for the lesson.
  • Select the link to that allows you to view the collaboration option for the lesson.
    Select the link "Collaboration Option" link to see the collaboration information.
  • What would you select to see how an assessment will be graded?
    Select the link for the "Grading Rubric" to see how an assessment is graded.
